Example of integrating Jonathan Oliver's EventStore into the Lokad CQRS project.

SETUP
---------------
1. Open server explorer, right click on Data Connections, Click on Create New Database, name the Database lokad-cqrs-eventstore, click OK.
2. Right click on the solution and select properties. Select "Multiple Startup Projects" and set Azure and Client to start.
3. Hit F5 on your keyboard.
4. Wait for the worker role to bootup.
5. Hit any key in the client window.
